# ev_range_prediction
Machine learning model to predict electric vehicle range using Linear Regression
# Electric Vehicle Range Prediction 🚗⚡

## Overview

This project predicts the driving range (km) of electric vehicles based on their technical specifications.

The goal of this project is to understand and implement Linear Regression for a real-world machine learning problem.

Two approaches were implemented:

1. Linear Regression from scratch using Gradient Descent
2. Linear Regression using Scikit-Learn


---

## Problem Statement

Electric vehicle range depends on several factors such as:

- Battery capacity
- Energy efficiency
- Vehicle weight
- Motor specifications
- Charging capability
- Vehicle dimensions

This project uses these specifications to predict the expected driving range of an electric vehicle.


---

## Dataset

The dataset contains electric vehicle specifications, including:

### Features

- Battery capacity (kWh)
- Top speed (km/h)
- Torque (Nm)
- Efficiency (Wh/km)
- Acceleration (0-100 km/h)
- Fast charging power (kW)
- Towing capacity (kg)
- Cargo volume (L)
- Vehicle dimensions
